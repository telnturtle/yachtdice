# Yacht Dice - 얏다이스

얏다이스(yacht dice, 요트 다이스)를 게임으로 구현해보자.

폰화면으로 세로로 플레이하는것을 전제함.


## 새로 배운것
- CSS linear-gradient
- CSS grid
- CSS 3D 애니메이션
- CSS `matrix3d()`
- CSS 그림자
